# Run and deploy your US payroll Agent app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1R5m3w6LnPHjKf6PcFCRtXbwub0DBqSAE

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
